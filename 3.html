Microsoft Excel

Microsoft excel is a software which is based on spreadsheets, using a grid of cells arranged in numbered rows and letter-named columns to organize data manipulations like arithmetic operations. It has a package of supplied functions to answer statistical, engineering and financial needs. In addition, it can display data as line graphs.You can see the column and cell number in a "Name box" and you can also see the formula on the "Formula Bar" The extention which is used for Microsoft Excel is ".xls" and its command key is"Excel"
